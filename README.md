# chatgpt-up-down-key-input-history

When you press the up or down key in chatgpt, a list of previously entered strings is applied to input.

![demo](./demo.gif)

## Build

pre-requires

- node.js
- pnpm

```shell
pnpm i
npm run build
# access chrome://extensions/
```
